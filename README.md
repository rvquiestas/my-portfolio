# My Website Portfolio

A personal portfolio website built using **React JS** and **Tailwind CSS** to showcase my projects, skills, and experiences.

## Live Preview
ðŸ”— [Visit My Portfolio](https://rvquiestas.vercel.app/)

## Tech Stack
- React JS  
- Tailwind CSS  
- Vite  
- Node.js (for backend, if any)
- Deployed with Vercel
- EmailJS (for contact form functionality)
- React Google reCaptcha (span protection for contact form)

## Features
- Responsive design across all devices  
- Dark and light mode toggle  
- Interactive project cards with live preview and source code links  
- Contact section with working form
